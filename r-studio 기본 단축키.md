## source 창 단축키
- 코드 실행 : ctrl + enter
- 소스 저장 : ctrl + s
- 처음부터 커서있는 곳까지 실행 : ctrl + alt + b
- source 창 이동 : ctrl + 1
- console 창 이동 : ctrl + 2
- source 파일 여러개 이동 시 : ctrl + alt + 왼/오 화살표
- 주석 처리 및 해제 : ctrl + shift + c
- 함수 및 R 소스파일 내용 보기 : F2
## console 창 단축키
- 코드 실행 : enter
- 실행된 코드 삭제 : ctrl + l
